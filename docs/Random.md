# random

### Example#1

```javascript
import JStr from "@akcybex/jstr";

const result = JStr.random(20);
console.log(result); // Outputs: 'JlPo591STBctjOjPw15d'
```

### Example#2

```javascript
import JStr from "@akcybex/jstr";

const result = JStr.random();

console.log(result); // Outputs: 'fEcu70M9LQ2Uknyk'
```
