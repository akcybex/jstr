# Repeat

### Example#1

```javascript
import JStr from "@akcybex/jsr";

const str = JStr.of("he").repeat(5).toString();
console.log(str); // Outputs: 'hehehehehe'
```

### Example#2

```javascript
import JStr from "@akcybex/jsr";

const str = JStr.repeat("a", 5);
console.log(str); // Outputs: 'aaaaa'
```
