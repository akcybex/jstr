# Upper

### Example#1

```javascript
import JStr from "@akcybex/jsr";

const str = JStr.of("hello").upper().toString();
console.log(str); // Outputs: 'HELLO'
```

### Example#2

```javascript
import JStr from "@akcybex/jsr";

const str = JStr.upper("hello").toString();
console.log(str); // Outputs: 'HELLO'
```
