# Lower

### Example#1

```javascript
import JStr from "@akcybex/jstr";

const str = JStr.of("HELLO").lower().toString();
console.log(str); // Outputs: 'hello'
```

### Example#2

```javascript
import JStr from "@akcybex/jstr";

const str = JStr.lower("HELLO");
console.log(str); // Outputs: 'hello'
```
